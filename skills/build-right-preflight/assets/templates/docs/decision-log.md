# Decision Log

Use this file for durable decisions that future agents or collaborators must
respect.

Record decisions about:

- MVP boundary
- source mode
- architecture choice
- deployment choice
- workflow customization
- stop-gate decisions

Do not record:

- routine command results
- transient implementation notes
- every file edit
- evidence that belongs in a task evidence log
